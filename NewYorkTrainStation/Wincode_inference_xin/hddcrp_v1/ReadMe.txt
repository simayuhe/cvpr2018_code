    控制台应用程序：hddcrp_v1 项目概述

应用程序向导已为您创建了此 hddcrp_v1 应用程序。

本文件概要介绍组成 hddcrp_v1 应用程序的每个文件的内容。


hddcrp_v1.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

hddcrp_v1.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

hddcrp_v1.cpp
    这是主应用程序源文件。

其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 hddcrp_v1.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

其他注释:
v1 这个版本只用来验证原程序（田国栋师兄留下的程序）是能够完整跑通的
v1.1 用来跑小数据集，并对一些函数的运行时间进行适当优化
v1.4 对节点消失情况，为指向它的节点重新采样链接
v1.5 使用rootnode leafnode对采样程序进行分化，主要原因是在v1.4中处理cur_stat时出现问题，需要在compute_margin时进行判断，所以进行重新整理

v1.6 在1.5的版本初步跑通的情况下，确认程序逻辑的正确性，并进行梳理，同样用小数据集验证正确性，然后移动到整个数据集上进行试验效果采集
